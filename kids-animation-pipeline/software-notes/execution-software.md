# Suggested Execution Software

## Best practical stack

### 1. Writing / planning
- OpenClaw agent — for idea generation, scripts, storyboards, QA, metadata
- Notion or Google Docs — for approval and tracking
- Airtable (optional) — for episode pipeline management

### 2. Voice generation
- ElevenLabs — best for expressive AI voices
- PlayHT — good alternative
- Human voice actors — best if you want maximum warmth and originality

### 3. 3D animation
- Blender — best free and flexible 3D tool
- Cinema 4D — easier in some motion workflows, but paid
- Reallusion iClone + Character Creator — faster character animation pipeline for non-experts

## My recommendation
If you want speed without a giant studio setup:
- Blender for environments and rendering
- Reallusion iClone for faster character animation
- ElevenLabs for temporary voice prototyping
- DaVinci Resolve for final editing

### 4. Asset creation
- BlenderKit — for reusable Blender assets
- Sketchfab — for 3D models and references
- Adobe Substance 3D Painter — for texturing if needed

### 5. Storyboarding / previs
- Canva — simple boards
- Figma — structured planning
- Blender Grease Pencil — for previs inside the 3D workflow

### 6. Editing / sound
- DaVinci Resolve — best all-around free editor
- Adobe Premiere Pro — if you already use Adobe
- Audacity — simple audio cleanup
- Adobe Audition — advanced audio cleanup

### 7. Music and sound effects
- Epidemic Sound
- Artlist
- YouTube Audio Library

## Recommended setup by budget

### Low budget
- OpenClaw
- Blender
- ElevenLabs starter tier
- DaVinci Resolve
- YouTube Audio Library

### Mid budget
- OpenClaw
- Blender + BlenderKit
- iClone + Character Creator
- ElevenLabs
- DaVinci Resolve Studio
- Epidemic Sound

### Higher speed setup
- OpenClaw
- iClone + Character Creator
- Blender for polish and environments
- ElevenLabs or human voice talent
- Premiere or DaVinci
- Artlist or Epidemic
