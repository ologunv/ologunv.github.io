# Agent Workflow

## Goal
Turn approved episode ideas into production-ready kids animation packs.

## Workflow Steps
1. Read `channel-bible/channel-bible.md`
2. Read `channel-bible/characters.md`
3. Read `channel-bible/world-rules.md`
4. Read the selected `episodes/ep-xxx/idea.md`
5. Generate:
   - `script.md`
   - `storyboard.md`
   - `voice-lines.md`
   - `animation-brief.md`
6. Check output against `templates/qa-checklist.md`
7. Revise for clarity, age fit, and production simplicity

## Rules for the Agent
- keep vocabulary simple for ages 4–6
- make scenes easy to follow
- use recurring characters consistently
- avoid unsafe or scary behavior
- use one clear lesson per episode
- keep endings positive and explicit

## Desired Output Format
For every approved episode, create these files:
- `script.md`
- `storyboard.md`
- `voice-lines.md`
- `animation-brief.md`

## Optional Next Layer
The workflow can later be extended to also generate:
- `youtube-package.md`
- `thumbnail-brief.md`
- `asset-prompts.md`
- `shot-timing.csv`
