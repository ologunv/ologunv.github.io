# Kids Animation Pipeline

A starter production system for a kids-friendly 3D animation YouTube channel.

## Recommended setup
- Audience: ages 4–6
- Format: recurring characters, one lesson per episode
- Runtime: 2.5–4 minutes
- Cadence: 2 videos per week
- Workflow: AI-assisted, human-approved

## Folder guide
- `channel-bible/` — brand, characters, world rules, lesson library, safety policy
- `templates/` — reusable production templates
- `episodes/` — one folder per episode
- `software-notes/` — tools and execution stack recommendations

## Production flow
1. Pick or approve an episode idea
2. Write script
3. Generate storyboard
4. Split voice lines
5. Prepare animation brief
6. Produce animation
7. Review for quality and policy
8. Package for YouTube upload
