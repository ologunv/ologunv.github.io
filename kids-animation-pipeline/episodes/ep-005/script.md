# Episode 005 Script — The Clean-Up Song Parade

## Lesson
Cleaning up can be fun when everyone helps together.

## Runtime Target
3 minutes

## Scene 1
- Setting: Playroom after playtime
- Visual action: Toys are scattered everywhere.
- Dialogue:
  - Lulu: "Wow... that's a big mess."
  - Bobo: "That's a lot of toys."
  - Mimi: "We can clean it up together."
- Narration: After a big day of play, the room was very messy.
- Music/SFX: Light playful music

## Scene 2
- Setting: Same playroom
- Visual action: Lulu starts a silly marching song and steps in rhythm.
- Dialogue:
  - Lulu: "Clean-up parade, hooray, hooray!"
  - Bobo: "A parade?"
  - Lulu: "Yes! March and tidy!"
- Narration: Lulu had a bright idea. What if cleaning up could feel like a parade?
- Music/SFX: Toy drum beat

## Scene 3
- Setting: Playroom zones
- Visual action: Friends each do a simple job.
- Dialogue:
  - Bobo: "I'll stack the blocks."
  - Mimi: "I'll sort the colors."
  - Tiko: "I'll put the books back neatly."
- Narration: Little by little, the room started to change.
- Music/SFX: Rhythmic clean-up song

## Scene 4
- Setting: Clean playroom
- Visual action: The room is tidy and everyone marches in one last circle.
- Dialogue:
  - Lulu: "We did it!"
  - Bobo: "The room looks great!"
  - Mimi: "Teamwork helped."
  - Tiko: "And the song helped too."
- Narration: Cleaning up felt much easier when everyone helped together.
- Music/SFX: Final happy chorus

## Ending Recap
- Lesson summary: Cleaning is easier and more fun when everyone helps.
- Final upbeat line: "Clean-up parade, hooray!"
