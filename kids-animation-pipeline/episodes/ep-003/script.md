# Episode 003 Script — Tiko Counts the Farm Friends

## Lesson
Counting can help us stay calm and solve small problems.

## Runtime Target
3 minutes

## Scene 1
- Setting: Sunny farmyard
- Visual action: Ducklings waddle around Tiko's feet.
- Dialogue:
  - Tiko: "I need to count the ducklings."
  - Lulu: "They're so wiggly!"
  - Bobo: "Let's count them!"
- Narration: Tiko had an important job. He needed to count the farm friends.
- Music/SFX: Farm ambience, soft quacks

## Scene 2
- Setting: Same farmyard
- Visual action: Ducklings keep moving and Tiko loses track.
- Dialogue:
  - Tiko: "One, two, three... oh!"
  - Tiko: "I lost count."
  - Bobo: "Let's go fast!"
  - Tiko: "Maybe slow is better."
- Narration: Counting was tricky when everyone kept waddling around.
- Music/SFX: Playful quacks and steps

## Scene 3
- Setting: Small fence line
- Visual action: Friends gently guide ducklings into a line.
- Dialogue:
  - Lulu: "One duckling here!"
  - Bobo: "And one over there!"
  - Tiko: "Now let's count slowly together."
- Narration: The friends worked together and made counting easier.
- Music/SFX: Gentle rhythm under counting

## Scene 4
- Setting: Farmyard, lined-up ducklings
- Visual action: Group counts from one to ten.
- Dialogue:
  - Group: "One, two, three, four, five, six, seven, eight, nine, ten!"
  - Tiko: "We did it!"
  - Tiko: "Slow and smart works too."
- Narration: Tiko showed that going slowly can be a smart way to get things right.
- Music/SFX: Happy success music

## Ending Recap
- Lesson summary: Counting slowly and calmly can help solve problems.
- Final upbeat line: "Let's count together!"
